О шоу

«Студия СОЮЗ» на ТНТ: спасите наши уши

Первое комедийно-музыкальное шоу!

Теперь звезды ТНТ будут мериться музыкальностью, чтобы сделать тебе очень смешно! Для этого каждый четверг им придется погружаться в мир изощренной русской эстрады, и только юмор и самоирония помогут им выстоять в этой схватке. Отныне четверги не будут прежними… Хотя бы потому, что в этом абзаце слово «четверг» повторяется целых пять раз!

Эксперименты

«ЛСДЕЦЛ»

Участникам предлагаются рисунки, в которых лица знаменитостей скрестили с каким-то предметом. Игрокам нужно произнести точный ответ, совместив увиденные элементы.

«Реп против политики»

Участникам предлагают цитаты, вынесенные вне контекста. Игрокам нужно определить, откуда взята цитата: из песни малоизвестного рэпера или из речи известного политика.

«Рифмобол»

Игрокам предлагают строчки из неизвестных песен, из которой убрали последние слова. Участникам необходимо выбрать правильную концовку из четырёх вариантов.

«Вы орете великолепно»

Участникам предлагается караоке с препятствиями, в котором надо управлять шариком с помощью громкости своего голоса. Выигрывает тот, кто допел песню до конца.

«Повторение мать его»

Участникам нужно повторить четверостишие, записанное заранее. В случае ошибки участника бьёт током. Побеждает тот, кто справится с заданием раньше соперника.

«Один раз не Костюшкин Стас»

Игрокам показывают песню, из которой с каждых двух строчек убираются по два слова. Каждый из игроков должен угадать, какое слово спрятано за одним из пропусков.
